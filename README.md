# Learning Repository

## Folders Overview

- **Numpy/** - NumPy fundamentals including array creation, data types, operations, and memory management
  - datatypes.py - Array creation methods, data types, views vs copies, and reshaping operations
- **Pandas/** - Pandas data manipulation and analysis with DataFrames, data access methods, and statistical operations
  - dataAccess.py - DataFrame creation, data selection, filtering, and basic operations
  - summary_statistics.py - Statistical analysis, aggregation, grouping, and data transformation
  - reshapingdata.py - Data combining, merging, and reshaping operations
