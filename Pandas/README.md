# Pandas Learning Repository

## Files Overview

### 1. `dataAccess.py`
**Description**: Basic data creation, file I/O, and data selection techniques
**Functions**: `pd.Series()`, `pd.DataFrame()`, `read_csv()`, `head()`, `info()`, `describe()`, `loc[]`, `iloc[]`, `notna()`, `isin()`

### 2. `order_data.py`
**Description**:An Excercise on unstructured Data cleaning and analysis of restaurant order data
**Functions**: `read_csv()`, `str.replace()`, `astype()`, `sort_values()`, `reset_index()`

### 3. `reshapingdata.py`
**Description**: Combining and merging multiple datasets
**Functions**: `concat()`, `merge()`, `groupby()`, `sort_values()`, `tail()`

### 4. `summary_statistics.py`
**Description**: Statistical analysis and data transformation operations
**Functions**: `dropna()`, `rename()`, `describe()`, `mean()`, `min()`, `max()`, `median()`, `agg()`, `groupby()`, `value_counts()`, `pivot_table()`, `pivot()`, `melt()`

## Quick Reference
- **Data Creation**: Series, DataFrame
- **File I/O**: CSV reading/writing
- **Selection**: loc, iloc, boolean indexing
- **Cleaning**: dropna, string operations
- **Analysis**: describe, groupby, pivot
- **Reshaping**: concat, merge, melt
